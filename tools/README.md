# tools/

## Purpose
Operational CLI tools for account and connectivity workflows.

## Key Files
- `polymarket_account.py`: Account utility CLI.
- `test_clob.py`: CLOB connectivity helper.
- `ws_test.py`: WebSocket connectivity helper.

## Usage
- `uv run -m tools.polymarket_account --help`

## Notes
Prefer these tools for ops tasks instead of ad-hoc scripts.
