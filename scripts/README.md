# scripts/

## Purpose
Ad-hoc/debug scripts for local experiments.

## Key Files
- `debug_*` scripts: local debugging helpers.
- `compare_prices.py`: price comparison utility.
- Other one-off helpers used during development.

## Usage
Run directly with `uv run python <script>` when needed.

## Notes
Not part of stable runtime API; keep production workflows in `make` targets and module CLIs.
